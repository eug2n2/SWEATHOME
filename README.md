# 🏠 SWEATHOME
홈 트레이닝을 위한 운동 콘텐츠 및 운동 기록 플랫폼 **SWEATHOME** 💪🏻

## 1️⃣ 역할 분담
- 🐶 **김윤**: 메인 프론트엔드, 서브 백엔드
- 🐰 **오유진**: 메인 백엔드, 서브 프론트엔드, DBA

## 2️⃣ 개발 일정
